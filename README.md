Financial Books is a lightweight React Native personal finance tracker with multi-account support, full CRUD, SQLite persistence, and user preferences for theme and formatting.

Features
- Add, edit, delete transactions.

- Multiple accounts with separate records.

- Search by keyword and filter by date range.

- Local SQLite storage.

- Dark mode and locale-aware number/date formatting.

- Create and restore backups for data export and recovery.

Avalaible for Android 7+.
